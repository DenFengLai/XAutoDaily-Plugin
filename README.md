# XAutoDaily-Plugin

> 本项目处于开发阶段十分不稳定(作者懒癌晚期没救了)
> 十分不建议安装，建议等作者懒癌痊愈后发布master分支后再安装使用
一个基于Yunzai-Bot的每日自动签到插件，灵感和插件名来自[XAutoDaily](https://github.com/LuckyPray/XAutoDaily)

## 配置方法

计划完全兼容锅巴进行配置，也可手动修改插件内`config`文件夹下配置文件

## 预计完工日期

1~70天(或更久)
